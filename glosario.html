Machine Learning: el arte de que los datos aprendan

Glosario

A continuación este glosario ha de aclarar cualquier duda sobre algunas palabras elegantes.

Algoritmo

Conjunto de instrucciones que le indican a una máquina cómo resolver un problema. En ML, el algoritmo aprende y mejora con los datos.

Dataset

Colección de datos que el modelo estudia, como si fueran sus “experiencias”. Mientras más variados y de calidad, mejor aprende.

Features (características)

Elementos específicos que describen los datos. Si analizas imágenes de gatos, las features pueden ser bordes, colores o formas.

Entrenamiento

Proceso donde el modelo aprende ajustándose a los datos, como un estudiante practicando hasta mejorar.

Inferencia

Momento en que el modelo ya entrenado usa su aprendizaje para tomar decisiones nuevas, sin intervención humana.

Overfitting

Cuando el modelo “memoriza” demasiado los ejemplos y falla al enfrentar datos nuevos. Es como estudiar solo las preguntas del examen en vez de aprender el tema.

Red neuronal

Modelo inspirado en el cerebro humano, compuesto por capas de nodos que procesan información y descubren patrones complejos.
